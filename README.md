# CPU 스케줄러

## 기술 스택
* IDE
```
 - Eclipse
```
* Language
```
 - Java
```
</br>

## 프로젝트 목적
* 다양한 스케줄링 알고리즘을 직접 구현해 봄으로써 각 스케줄링 기법의 동작 과정을 이해한다.

* 각 스케줄링 알고리즘의 간트차트를 분석하여 특성을 파악한다.

* 구현한 프로그램을 통해 모의실험을 함으로써 각 스케줄링 알고리즘을 평가해 본다.
</br>

## 프로젝트 기능
* 입력받은 데이터에 대한 총 실행 시간, 평균 대기시간, 평균 반환시간을 출력

* 스케줄링 과정을 간트 차트로 출력
</br>
